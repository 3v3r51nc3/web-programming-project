# EventHub Frontend

This folder is a cleaned React starter for the assignment.

Vite is kept only as the local development tool. The assignment requires a React SPA, and this setup satisfies that requirement without adding unnecessary setup work.

## Current State

- minimal React app
- simple clean CSS
- ready to expand into the real assignment pages

## Next Pages To Build

- Login
- Dashboard
- Events
- Event details
- Participants

## Environment

Use `VITE_API_BASE_URL` later when the Django backend is ready.

Default planned value:

```bash
VITE_API_BASE_URL=http://localhost:8000/api
```

## Run

```bash
npm install
npm run dev
```
